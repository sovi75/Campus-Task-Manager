# User Manual
## Guide to Using Campus Task Manager

Welcome to the **Campus Task Manager**! This manual outlines the main features of the application and details how to navigate the system to keep track of your academic deadlines.

---

## 1. Getting Started

### 1.1 Creating an Account
1. Open the application in your browser. You will be redirected to the **Login** screen.
2. Click **Create an account** at the bottom of the card.
3. Fill in your **Full Name**, **Email Address** (must be a valid format), and a secure **Password** (minimum of 6 characters).
4. Click **Sign Up** to create your account. You will be logged in automatically and redirected to the **Dashboard**.

### 1.2 Logging In
1. Enter your registered email and password on the **Login** screen.
2. Click **Sign In**.
3. If the credentials match, you will be redirected to your personal **Dashboard**.

---

## 2. Navigating the Dashboard

The **Dashboard** serves as the central hub for your tasks and study statistics.

```text
+-------------------------------------------------------------------------+
|  CAMPUS TASK MANAGER                                  [+] New Task      |
+-------------------------------------------------------------------------+
|  (Sidebar)      |  OVERVIEW WIDGETS                                     |
|  - Dashboard    |  [ Total Tasks ] [ Pending ] [ Completed ] [ Overdue ] |
|  - All Tasks    |                                                       |
|  - Study Mode   |  TODAY'S FOCUS             UPCOMING DEADLINES         |
|  - Calendar     |  * Task 1                  * Task 2                   |
|                 |  * Task 3                  * Task 4                   |
|  QUICK FILTERS  |                                                       |
|  - Pending      |  INLINE QUICK ADD FORM                                |
|  - Completed    |  [ Enter task title...  ] [Due Date] [ Category ] (+) |
+-------------------------------------------------------------------------+
```

### 2.1 Overview Widgets
- The top row displays statistics for **Total Tasks**, **Pending Tasks**, **Completed Tasks**, and **Overdue Tasks**. Clicking these widgets opens the filtered view.

### 2.2 Today's Focus & Upcoming Deadlines
- **Today's Focus**: Lists tasks that are due today.
- **Upcoming Deadlines**: Lists upcoming tasks, sorted chronologically so you can see what is due next.

### 2.3 Inline Quick Add Form
- Create tasks quickly by entering a title, selecting a due date, picking a category (Assignment, Exam, Project, Personal), and clicking **Quick Add**.

---

## 3. Managing Your Tasks

Click **All Tasks** in the sidebar to access the full task manager.

### 3.1 Task Table
- View all your tasks, each showing a course code, priority badge (High, Medium, Low), category, due date, status, and edit/delete actions.

### 3.2 Filtering & Searching
- **Search Box**: Type key terms (e.g. "Math", "Lab") to filter the list by title or description.
- **Status Tabs**: Filter the list by **All**, **Pending**, or **Completed** status.
- **Priority Filter**: Filter the list by priority levels (e.g., show only High priority tasks).
- **Category Filter**: Filter the list by category tags (e.g., show only Exams).
- **Sort Dropdown**: Sort tasks by due dates ascending/descending, priority, or creation date.

### 3.3 Task Details Panel
- Click a task card to open the **Detail Side Panel**. The panel displays the task description, reminders configuration, and status.

### 3.4 Creating & Editing Tasks (Modal)
- Click **New Task** in the top header (or edit an existing task) to open the task editor dialog. Here you can configure the task name, description, priority, category, course code, and reminder preferences.

### 3.5 Bulk Actions
- Select multiple tasks using the check boxes on the left side of each task card.
- Use the **Bulk Action Bar** at the bottom to mark the selected tasks as completed or delete them all at once.

---

## 4. Setting Task Reminders

1. Open the task creator/editor modal.
2. Select **Enable Reminder** to open the options dropdown.
3. Choose when you want to receive the reminder: **At time of due**, **1 hour before**, or **1 day before**.
4. Save the task.
5. If prompted by your browser, select **Allow** to enable desktop notifications.

---

## 5. Pomodoro Study Mode

Click **Study Mode** in the sidebar to open the focus page.

### 5.1 Pomodoro Timer
- Click the timer modes (**Pomodoro Focus**, **Short Break**, or **Long Break**) to change the timer duration.
- Click **Start Focus** to start the timer.
- Click **Pause** or **Reset** to control the timer.

### 5.2 Synthesized Ambient Noise
- Click play (**▶**) next to any sound option (**Rainfall**, **Lo-Fi Beats**, **Cafe Ambience**, **White Noise**) to generate background noise.
- Click pause (**⏸**) to stop the sound generator.

### 5.3 Active Focus Task
- Use the dropdown to select a task from your list to work on during your focus session.
- Once the Pomodoro timer runs out, the app will ask if you want to mark the selected task as completed.

---

## 6. Interactive Calendar Sync

Click **Calendar Sync** in the sidebar to open the monthly calendar grid.

### 6.1 Calendar Grid
- Click **◀** or **▶** next to the month header to navigate between months.
- Click **Today** to return to the current month.
- Days with deadlines display colored labels showing task names. Green tags indicate completed tasks.

### 6.2 Selected Day Tasks
- Click any day in the grid to view all tasks due on that date in the sidebar list. You can mark tasks completed directly from this list using the check boxes.

### 6.3 Syncing Calendars
- Click **Sync Calendar Now** to run a simulated calendar sync. A progress bar will show the sync status as it connects to Google Calendar and Outlook.
- Once completed, the card will display the date and time of the last sync.
